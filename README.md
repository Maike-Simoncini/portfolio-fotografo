# 📸 Portfólio de Fotógrafo

Um portfólio profissional desenvolvido com **HTML5** e **CSS3**, projetado para fotógrafos que desejam apresentar seus trabalhos de forma elegante, moderna e impactante.

O projeto combina uma estética minimalista com animações suaves, efeitos cinematográficos e uma experiência visual premium, proporcionando uma navegação sofisticada para clientes e visitantes.

---

## ✨ Funcionalidades

- 🎨 Design moderno e minimalista
- 📱 Layout totalmente responsivo
- 🖱 Cursor personalizado interativo
- 🎬 Efeito Film Grain (granulação cinematográfica)
- ⚡ Animações suaves e performáticas
- 📷 Galeria de fotos estilo Masonry Grid
- 👤 Seção Sobre personalizada
- 📩 Formulário de contato elegante
- 🔗 Integração com redes sociais
- 🎯 Navegação com Smooth Scroll
- 🌙 Visual premium inspirado em portfólios de luxo

---

## 🖼️ Seções do Projeto

### Hero Section

Apresentação principal do fotógrafo com:

- Nome em destaque
- Frase de impacto
- Redes sociais
- Indicador de rolagem
- Elementos visuais decorativos

### Galeria

Exibição dos trabalhos através de:

- Layout assimétrico
- Hover Effects
- Categorias de fotografia
- Estrutura dinâmica

### Sobre

Área destinada para:

- História do fotógrafo
- Experiência profissional
- Estatísticas e conquistas

### Contato

Formulário para:

- Solicitação de orçamento
- Agendamento de ensaios
- Contato direto com clientes
